# SentinelAI Frontend

React 18 + TypeScript + Recharts dashboard for the SentinelAI social media monitoring platform.

## Run
```bash
npm install
npm run dev
# Opens at http://localhost:3000
```

## Features
- Real-time mention feed via WebSocket
- Sentiment trend charts (Recharts)
- Brand Health Score
- Ticket Kanban board
- Reply approval queue
- Agent activity log
- Analytics dashboard
